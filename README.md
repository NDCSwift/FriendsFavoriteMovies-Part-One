# 🎬 Friends Favorite Movies — Part 1
A SwiftUI multi-screen app series — Part 1 sets up the data model and home screen for a friend group movie tracker.

---

## 🤔 What this is
This is the first part of a multi-video series building a SwiftUI app where friends can track and share their favorite movies. Part 1 covers creating the data model, setting up the main list view with sample data, and navigating to a detail screen. A great beginner-friendly project that grows across episodes.

## ✅ Why you'd use it
- **SwiftUI list & navigation** — Shows how to build a List with NavigationStack and push to a detail view
- **Data modeling** — Demonstrates creating a simple Swift struct model and populating views with it
- **Series foundation** — The starting point for a multi-part project — follow along to see it grow into a full app

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/FriendsFavoriteMovies-Part-One.git
cd FriendsFavoriteMovies-Part-One
```

### 2. Open in Xcode
Double-click `FriendsFavoriteMovies1.xcodeproj`.

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team** — select your team.

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique reverse-domain ID.

## 🛠️ Notes
- This is Part 1 of a series — check the channel for Parts 2 and the Final version.
- If you see a code signing error, verify Team and Bundle ID are set.
